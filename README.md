# ContactsManager

Programme Java de gestion de contacts.
Permet d'ajouter des contacts (nom + numéro de téléphone ivoirien)
et de rechercher un contact par son nom.

## Fonctionnalités
- Ajouter un contact
- Rechercher un contact par nom
- Stocker jusqu'à 500 contacts

## Comment exécuter
1. Cloner le dépôt
2. Ouvrir le projet dans IntelliJ
3. Exécuter la classe Main
